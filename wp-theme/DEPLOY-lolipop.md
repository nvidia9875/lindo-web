# LINDO テーマ — ロリポップ導入手順

このテーマ（`wp-theme/lindo/`）を、ロリポップの WordPress に載せて公開するまでの手順です。
専門用語は最小限にしています。上から順にやれば公開できます。

---

## 0. 全体像（5ステップ）

1. ロリポップに WordPress を入れる（ドメインは既存のものを使い回す）
2. テーマ `lindo` をアップロードして有効化
3. プラグイン **Contact Form 7** と **Flamingo** を入れる
4. アーティストを登録 → 問い合わせフォームを設定
5. 仕上げ（SSL・セキュリティヘッダー・旧ページの非公開）

---

## 1. WordPress を入れる（ドメイン使い回し）

ロリポップには「**WordPress 簡単インストール**」があります。

1. ロリポップ・ユーザー専用ページ → **サイト作成ツール → WordPress 簡単インストール**
2. **サイトURL**：使い回したい既存ドメインを選ぶ
   - ドメインがまだロリポップに紐づいていない場合は、先に
     **独自ドメイン設定**でドメインを追加（ムームードメイン管理なら自動で楽）。
   - サブディレクトリは空（＝ドメイン直下に出す）が基本。
3. データベースは「新規自動作成」でOK。
4. 管理者ユーザー名・パスワード・メールを控える。
5. インストール後 **独自SSL（無料）** を必ず有効化（後述のセキュリティに必要）。
   - ユーザー専用ページ → **セキュリティ → 独自SSL証明書導入** → 対象ドメインを「SSL保護有効」に。
6. WordPress 管理画面（`https://ドメイン/wp-admin/`）にログイン。

> すでに同ドメインで別サイトが動いている場合は、移行になります。
> その時は事前にバックアップを取り、別サブディレクトリで作ってから切替えると安全です。

---

## 2. テーマをアップロードして有効化

1. `wp-theme/lindo/` フォルダを **zip 圧縮**して `lindo.zip` を作る。
   - macOS: `lindo` フォルダを右クリック →「"lindo" を圧縮」。
   - ⚠ `preview/` や `htaccess-sample.txt` は **テーマには含めない**（`lindo/` の中だけを zip）。
2. WordPress 管理画面 → **外観 → テーマ → 新規追加 → テーマのアップロード**
3. `lindo.zip` を選んでインストール → **有効化**。
4. **設定 → 表示設定 → ホームページの表示** を「**固定ページ**」にし、
   トップ用の空の固定ページ（例「Home」）を作って指定。
   - これで `front-page.php`（Hero / About / Service / Artists / Partner / Contact）が
     トップに出ます。
5. **設定 → パーマリンク** を開いて「変更を保存」を一度押す（URL設定の反映）。

---

## 3. 問い合わせ：Contact Form 7 + Flamingo

「**メールで届く＋管理画面にも溜まる**」を両立する定番構成です。

1. **プラグイン → 新規追加** で以下を検索して導入・有効化：
   - **Contact Form 7**（フォーム本体・メール送信）
   - **Flamingo**（送信内容を管理画面に蓄積。後から一覧・CSV書き出し可）
2. **お問い合わせ → コンタクトフォーム** で新規フォームを作成。
   「フォーム」タブに下記を貼り付け（テーマのデザインに合う構造）：

   ```
   <p class="field">
     <label>お名前 / 会社名 <span class="req">*</span>
       [text* your-name placeholder "株式会社○○ ／ 山田 太郎"]</label>
   </p>
   <p class="field">
     <label>メールアドレス <span class="req">*</span>
       [email* your-email placeholder "you@example.com"]</label>
   </p>
   <p class="field">
     <label>ご相談の種類
       [select your-subject "ビジュアルクリエイティブ" "ブランディング" "スタイリング" "その他"]</label>
   </p>
   <p class="field">
     <label>メッセージ <span class="req">*</span>
       [textarea* your-message placeholder "ご依頼内容・ご予算・希望時期など"]</label>
   </p>
   <p class="field">[submit class:btn "Send Message"]</p>
   ```

3. 「**メール**」タブで送信先を設定：
   - **送信先**：`contact@styledbylindo.com`（または運用中の受信用アドレス）
   - **送信元**：`wordpress@ドメイン` のような“同一ドメインのアドレス”にすると届きやすい
     （ロリポップでメールアドレスを作って使うと確実）。
   - 件名・本文は適宜。`[your-name]` などのタグで内容を差し込める。
4. 保存すると上部に **ショートコード**（例 `[contact-form-7 id="123" title="お問い合わせ"]`）が出る。コピー。
5. **外観 → カスタマイズ → LINDO — Contact**：
   - 「**Contact Form 7 ショートコード**」に 4. のショートコードを貼る。
   - 「**直接連絡用メールアドレス**」を確認（既定 `contact@styledbylindo.com`）。
   - 公開。これで Contact 欄が CF7 フォームに差し替わります（未設定の間はデザイン確認用の静的フォームが出ます）。

> 迷惑メール対策：CF7 の設定でロリポップのメールアドレスを送信元にする／
> 必要なら reCAPTCHA や Akismet を後から追加できます（ハニーポット系プラグインでも可）。

---

## 4. アーティストを登録

1. 管理画面の左メニュー **アーティスト → 新規追加**。
2. **タイトル**：表示名（例 `AURORA`）。
3. **アイキャッチ画像**：ポートレート（縦長 4:5 推奨。自動で 900×1125 にトリミング）。
4. 右下「アーティスト情報」：
   - サブ名 / かな（例 `オーロラ`）
   - 役職・カテゴリ（例 `Visual Creative / Styling` ← スラッシュ区切りでタグになる）
   - プロフィール（空行で段落が分かれる）
   - Instagram / X / YouTube / Website の URL（入れたものだけリンク表示）
5. 「**Instagram風グリッド画像**」：
   - 「画像を選択 / 追加」で複数選択（正方形に切り出し・3列表示）。
   - サムネイルを**ドラッグで並び替え**、不要は「×」。
6. **並び順**：右側「ページ属性 → 順序」に数値（小さいほど先）。
7. 公開。トップの Artists に出る → カードを押すとモーダルで詳細＋グリッド。

> 画像は表示サイズより極端に大きい元画像を上げない（軽さ優先）。
> 1辺 1200px 程度で十分。WebP 化プラグイン（EWWW など）を入れると更に軽くなります。

---

## 5. 仕上げ

### セキュリティヘッダー
- `htaccess-sample.txt` の内容を、WordPress ルートの `.htaccess` に
  「# END WordPress」より**下**へ追記（**SSL有効化後**に）。

### 旧ページの非公開
- 以前の静的デモ（`/concepts/`、`/works/` 等）は今回の WordPress には載せません。
  - 旧 Vercel 等で公開していたものがあれば停止、または `noindex` のまま放置でOK。
  - もし同ドメインの実ファイルとして残る場合は削除 or アクセス制限。

### 表示確認
- スマホ実機で Hero・Artists（カード→モーダル）・Contact 送信までを一度通す。
- 問い合わせを 1 通テスト送信し、**メール受信**と **Flamingo（お問い合わせ→受信メッセージ）への蓄積**の両方を確認。

### ロゴ・フォント・本番画像
- 実ロゴ（SVG）があれば **外観 → カスタマイズ → サイト基本情報 → ロゴ** で設定可
  （未設定時は `LINDO` テキストロゴ）。
- 本番のアーティスト画像に差し替え。

---

## 困ったとき
- フォームのメールが届かない → 送信元アドレスを「同一ドメインのメール」に変更／ロリポップのメール作成。
- レイアウトが崩れる → 「設定 → パーマリンク」を保存し直す／ブラウザのスーパーリロード。
- 画像が出ない → アイキャッチ未設定／グリッド未選択を確認。
